NAME
HelpxNS::EnterShortDescription — Enter in a short description of the command (<= 60 chars).
SYNOPSIS
HelpxNS::EnterShortDescription Command ShortDescription
DESCRIPTION
Return Value
ID of entry used in the commandinfo table.
Arguments
CommandCommand name or ID to enter short description for.
ShortDescriptionShort description (<= 60 chars) of the command.
Error Conditions
ConditionError CodeError Message
Input value is empty. -12 Variable $VarName has empty value.
EXAMPLES 
% HelpxNS::EnterShortDescription SampleCommand {Short description here.}
1
% HelpxNS::ShortDescriptionFor SampleCommand
Short description here.
