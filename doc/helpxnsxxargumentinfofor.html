NAME
HelpxNS::ArgumentInfoFor — Get dict with argument names and descriptions for command.
SYNOPSIS
HelpxNS::ArgumentInfoFor Command
DESCRIPTION
Return Value
None.
Arguments
CommandName or ID of command to get argument info for.
EXAMPLES 
% HelpxNS::EnterArgumentInfo SampleCommand Arg1 {The first argument.}
1
% HelpxNS::EnterArgumentInfo SampleCommand Arg2 {The second argument.} 
2
% HelpxNS::ArgumentInfoFor SampleCommand
Arg1 {The first argument.} Arg2 {The second argument.}
SEE ALSO
HelpxNS::EnterArgumentInfo, HelpxNS::EraseArgumentInfo
