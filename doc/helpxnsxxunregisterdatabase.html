NAME
HelpxNS::UnregisterDatabase — Unregister a database connection from helpx.
SYNOPSIS
HelpxNS::UnregisterDatabase ConnectionName
DESCRIPTION
Return Value
None.
Arguments
ConnectionNameName of database connection to unregister.
Error Conditions
ConditionError CodeError Message
Input value is empty. -12 Variable $VarName has empty value.
More Info
Once a connection is unregistered, helpx will no longer try to use that database connection to search for info about commands.

This command will not complain if you try to unregister a database that is not on the list. Currently, there is no option to force an error to be thrown in that scenario.

EXAMPLES 
% sqlite3 sampledb sample.db
% HelpxNS::RegisterDatabase sampledb
% HelpxNS::UnregisterDatabase sampledb
