NAME
HelpxNS::EnterMoreInfo — Enter what is to appear in the More Info section.
SYNOPSIS
HelpxNS::EnterMoreInfo Command MoreInfo
DESCRIPTION
Return Value
ID of entry used in the commandinfo table.
Arguments
CommandID or name of the command to enter more info for.
MoreInfoWhat is to go in the More Info section.
EXAMPLES 
% HelpxNS::EnterMoreInfo SampleCommand {Additional details.}
1
% HelpxNS::MoreInfoFor SampleCommand
Additional details.
