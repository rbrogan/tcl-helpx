About Helpx

Warning!

Helpx is currently in a pre-release state. It is made available now for purposes of accessing help documentation for the Gen project from the CLI.

What is Helpx for?

Helpx is for generating and accessing help information in various ways (CLI, HTML pages, .chm file).

How does Helpx work now?

For helpx-cli, each package that uses it will have its own help database file. Each package will open a database connection to that database file and register that connection with Helpx. This is what the Gen package does when it is loaded. After that, when you call helpx from the command line, it will search that database for help info and then format it for the terminal and print it out.

How will Helpx work in the future?

After the next release you will be able to populate the help database mentioned in the previous section. You could then bundle helpx with your packages and give your users CLI access to help info.

In future releases, that same database will be used for things like generating HTML pages. Likewise, we anticipate adding more advanced search capabilities.
