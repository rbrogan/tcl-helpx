NAME
HelpxNS::CreateCommand — Create a new command entry.
SYNOPSIS
HelpxNS::CreateCommand Name
DESCRIPTION
Return Value
ID of newly created command.
Arguments
NameName of the command to create.
Error Conditions
ConditionError CodeError Message
Input value is empty. -12 Variable $VarName has empty value.
Value already exists. -18 Value $Value in variable $VarName already exists.
More Info
Note that command names are not required to be unique.
EXAMPLES 
% HelpxNS::CreateCommand SampleCommand
1
SEE ALSO
HelpxNS::DeleteCommand, HelpxNS::RenameCommand
