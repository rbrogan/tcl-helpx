NAME
HelpxNS::EraseSeeAlso — Erase see also link, at given position.
SYNOPSIS
HelpxNS::EraseSeeAlso Command Position
DESCRIPTION
Return Value
None.
Arguments
CommandCommand name or ID.
PositionPosition of the argument, starting from 0-index.
Error Conditions
ConditionError CodeError Message
Contents of variable are invalid. -11 Variable $VarName has invalid value $Value.
Input value is empty. -12 Variable $VarName has empty value.
Input value is out-of-range. -8 Input value $Value is out-of-range.
More Info
Note that the position of other examples will be automatically adjusted, like if you were deleting an element from a list.
EXAMPLES 
% HelpxNS::EnterSeeAlso SampleCommand AnotherCommand
1
% HelpxNS::EnterSeeAlso SampleCommand AdditionalCommand
2
% puts [HelpxNS::SeeAlsoFor SampleCommand]
AnotherCommand AdditionalCommand
% HelpxNS::EraseSeeAlso SampleCommand 0
% puts [HelpxNS::SeeAlsoFor SampleCommand]
AdditionalCommand
SEE ALSO
HelpxNS::EnterSeeAlso, HelpxNS::SeeAlsoFor
