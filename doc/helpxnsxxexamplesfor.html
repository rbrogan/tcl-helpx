NAME
HelpxNS::ExamplesFor — Get an ordered list of examples showing command in use.
SYNOPSIS
HelpxNS::ExamplesFor Command
DESCRIPTION
Return Value
List where each element is an example usage of the command.
Arguments
CommandName or ID of command to get examples of.
EXAMPLES 
% HelpxNS::EnterExample SampleCommand {% SampleCommand 1 2\n1 2}
1
% HelpxNS::ExamplesFor SampleCommand
{% SampleCommand 1 2\n1 2}
SEE ALSO
HelpxNS::EnterExample, HelpxNS::EraseExample
KEYWORDS
