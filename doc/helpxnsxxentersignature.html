NAME
HelpxNS::EnterSignature — Enter signature of the command.
SYNOPSIS
HelpxNS::EnterSignature Command Signature
DESCRIPTION
Return Value
ID of entry used in the commandinfo table.
Arguments
CommandCommand name or ID.
SignatureSignature for the command.
Error Conditions
ConditionError CodeError Message
Input value is empty. -12 Variable $VarName has empty value.
More Info
Currently this command makes no check to see whether the signature is well-formed.
EXAMPLES 
% HelpxNS::EnterSignature SampleCommand {SampleCommand {Arg1 Arg2}}
1
% HelpxNS::SignatureFor SampleCommand
SampleCommand {Arg1 Arg2}
