NAME
HelpxNS::EnterArgumentInfo — Insert or update argument info for a command.
SYNOPSIS
HelpxNS::EnterArgumentInfo Command ArgName ArgDesc ?Position?
DESCRIPTION
Return Value
ID of argument info entry created or updated in the commandinfo table.
Arguments
CommandID or name of command to enter argument info for.
ArgNameName of argument.
ArgDescDescription of argument.
PositionOptional. Order of argument, starting from zero index. If left blank, argument will be added at the end.
Error Conditions
ConditionError CodeError Message
Contents of variable are invalid. -11 Variable $VarName has invalid value $Value.
Input value is empty. -12 Variable $VarName has empty value.
Input value is empty. -12 Variable $VarName has empty value.
Value already exists. -18 Value $Value in variable $VarName already exists.
More Info
If you give a position beyond the last position then a new element will be created at the end.

If you want to change an existing argument description, then you do not have to specify the position. The command will look up the position for the argument with the given name and overwrite the argument description.

If you enter an argument with the same name as another argument and a different position then the command will generate an error and complain of a duplicate argument.

Currently there is no way to move arguments up and down in position. You must first erase the argument and then re-enter it.

EXAMPLES 
% HelpxNS::EnterArgumentInfo SampleCommand Arg1 {The first argument.}
1
% HelpxNS::EnterArgumentInfo SampleCommand Arg2 {The second argument.}
2
% HelpxNS::ArgumentInfoFor SampleCommand
Arg1 {The first argument.} Arg2 {The second argument.}
