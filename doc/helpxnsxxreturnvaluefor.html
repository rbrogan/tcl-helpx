NAME
HelpxNS::ReturnValueFor — Get description of the return value of the given command.
SYNOPSIS
HelpxNS::ReturnValueFor Command
DESCRIPTION
Return Value
String containing a description of the return value for the given command.
Arguments
CommandName or ID of command to get return value description of.
EXAMPLES 
% HelpxNS::EnterReturnValueFor SampleCommand None.
% HelpxNS::ReturnValueFor SampleCommand
None.

HelpxNS::EnterReturnValue
