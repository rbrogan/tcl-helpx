NAME
HelpxNS::InfoDataFor — Get piece of information about command of certain type.
SYNOPSIS
HelpxNS::InfoDataFor Command InfoType
DESCRIPTION
Return Value
String with requested information.
Arguments
CommandName or ID of command to get piece of information about.
InfoTypeType of information to get.

It is possible to get the following types of information:
