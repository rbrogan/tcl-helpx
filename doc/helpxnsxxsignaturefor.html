NAME
HelpxNS::SignatureFor — Get the name and arguments for the given command.
SYNOPSIS
HelpxNS::SignatureFor Command
DESCRIPTION
Return Value
A string containing the name and arguments for the given command (see example).
Arguments
CommandName or ID of command to get signature for.
EXAMPLES 
% HelpxNS::EnterSignature SampleCommand {SampleCommand {Arg1 Arg2}}
1
% HelpxNS::SignatureFor SampleCommand
SampleCommand {Arg1 Arg2}
SEE ALSO
HelpxNS::EnterSignature
