NAME
HelpxNS::RegisterDatabase — Register a documentation database with helpx.
SYNOPSIS
HelpxNS::RegisterDatabase ConnectionName
DESCRIPTION
Return Value
None.
Arguments
ConnectionNameName of database connection to register.
Error Conditions
ConditionError CodeError Message
Input value is empty. -12 Variable $VarName has empty value.
More Info
Note that the caller is responsible for opening and closing the connection, and also unregistering it, if need be.

If the same connection name is registered twice, it is not an error, but rather will be silently ignored. The caller should take care that the connection name is sufficiently distinctive.

If you are getting an 'invalid command name' error then probably (1) connection was never set up or (2) connection was closed.

Note can you can access this list variable as HelpxNS::DatabaseConnectionList.

EXAMPLES 
% sqlite3 sampledb sample.db
% HelpxNS::RegisterDatabase sampledb
