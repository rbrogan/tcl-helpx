NAME
HelpxNS::RenameCommand — Rename the command to a new name.
SYNOPSIS
HelpxNS::RenameCommand Command NewName
DESCRIPTION
Return Value
None.
Arguments
CommandCommand name or ID.
NewNameNew name for command.
Error Conditions
ConditionError CodeError Message
Input value is empty. -12 Variable $VarName has empty value.
Value already exists. -18 Value $Value in variable $VarName already exists.
EXAMPLES 
% HelpxNS::CreateCommand SampleCommand
% HelpxNS::EnterShortDescription SampleCommand {Short description here}
% HelpxNS::ShortDescriptionFor SampleCommand
Short description here.
% HelpxNS::RenameCommand SampleCommand MyCommand
% HelpxNS::ShortDescriptionFor MyCommand
Short description here.
