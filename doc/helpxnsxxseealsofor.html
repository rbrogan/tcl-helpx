NAME
HelpxNS::SeeAlsoFor — Get a list of commands related to the given command.
SYNOPSIS
HelpxNS::SeeAlsoFor Command
DESCRIPTION
Return Value
A list of commands related to the given command.
Arguments
CommandName or ID of command to get see also information for.
More Info
List will be in alphabetical order.
EXAMPLES 
% HelpxNS::EnterSeeAlso SampleCommand {AnotherCommand AdditionalCommand}
1
% HelpxNS::SeeAlsoFor SampleCommand
AnotherCommand AdditionalCommand
SEE ALSO
HelpxNS::EnterSeeAlso
