NAME
HelpxNS::EnterReturnValue — Create / update a return value description for command.
SYNOPSIS
HelpxNS::EnterReturnValue Command ReturnValueDescription
DESCRIPTION
Return Value
ID of the entry used in the commandinfo table.
Arguments
CommandID or name of command to enter return value description for.
ReturnValueDescriptionDescription of return value for command.
EXAMPLES 
% HelpxNS::EnterReturnValue SampleCommand None.
1
% HelpxNS::ReturnValueFor SampleCommand
None.

HelpxNS::ReturnValueFor
