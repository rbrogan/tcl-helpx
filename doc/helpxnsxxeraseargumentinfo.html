NAME
HelpxNS::EraseArgumentInfo — Delete argument and adjust positions.
SYNOPSIS
HelpxNS::EraseArgumentInfo Command Position
DESCRIPTION
Return Value
None.
Arguments
CommandCommand name or ID.
PositionPosition of the argument, starting from 0-index.
Error Conditions
ConditionError CodeError Message
Contents of variable are invalid. -11 Variable $VarName has invalid value $Value.
Input value is empty. -12 Variable $VarName has empty value.
Input value is out-of-range. -8 Input value $Value is out-of-range.
More Info
Note that the position of other examples will be automatically adjusted, like if you were deleting an element from a list.
EXAMPLES 
% HelpxNS::EnterArgumentInfo SampleCommand Arg1 {The first argument.}
1
% HelpxNS::EnterArgumentInfo SampleCommand Arg2 {The second argument.}
2
% HelpxNS::ArgumentInfoFor SampleCommand
Arg1 {The first argument.} Arg2 {The second argument.}
% HelpxNS::EraseArgumentInfo SampleCommand 1
% HelpxNS::ArgumentInfoFor SampleCommand
Arg1 {The first argument.}
