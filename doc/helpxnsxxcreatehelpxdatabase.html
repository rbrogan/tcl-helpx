NAME
HelpxNS::CreateHelpxDatabase — Create a database file and populate it with helpx tables.
SYNOPSIS
HelpxNS::CreateHelpxDatabase FilePath
DESCRIPTION
Return Value
None.
Arguments
FilePathPath to create database at.
Error Conditions
ConditionError CodeError Message
Input value is empty. -12 Variable $VarName has empty value.
More Info
For more information about the database tables that are created, see Helpx Database Schema.
EXAMPLES 
% HelpxNS::CreateHelpxDatabase ./sample-helpx.db
