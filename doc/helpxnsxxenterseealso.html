NAME
HelpxNS::EnterSeeAlso — Enter other commands to make links to.
SYNOPSIS
HelpxNS::EnterSeeAlso Command SeeAlsoData ?Position?
DESCRIPTION
Return Value
ID of entry used in the commandinfo table.
Arguments
CommandCommand name or ID.
SeeAlsoDataCommand to link to.
PositionOptional. Specify where the link is to appear in the order. Defaults to put it at the end.
Error Conditions
ConditionError CodeError Message
Contents of variable are invalid. -11 Variable $VarName has invalid value $Value.
Input value is empty. -12 Variable $VarName has empty value.
EXAMPLES 
% HelpxNS::EnterSeeAlso SampleCommand {AnotherCommand AdditionalCommand}
1
% HelpxNS::SeeAlsoFor SampleCommand
AnotherCommand AdditionalCommand
SEE ALSO
HelpxNS::SeeAlsoFor
