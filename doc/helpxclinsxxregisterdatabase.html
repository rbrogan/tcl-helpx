NAME
HelpxCliNSxxRegisterDatabase — Register a documentation database with helpx.
SYNOPSIS
HelpxCliNS::RegisterDatabase ConnectionName
DESCRIPTION
Return Value
None.
Arguments
ConnectionNameName of database connection to register.
More Info
Note that the caller is responsible for opening and closing the connection, and also unregistering it, if need be.

If the same connection name is registered twice, it is not an error. The caller should take care that the connection name is sufficiently distinctive.

If you are getting an 'invalid command name' error then probably (1) connection was never set up or (2) connection was closed.

EXAMPLES 
% sqlite3 sampledb sample.db
% HelpxNS::RegisterDatabase sampledb
