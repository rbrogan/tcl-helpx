NAME
HelpxNS::MoreInfoFor — Get additional information for given command.
SYNOPSIS
HelpxNS::MoreInfoFor Command
DESCRIPTION
Return Value
String with additional information for the given command.
Arguments
CommandName or ID of command to more information about.
EXAMPLES 
% HelpxNS::EnterMoreInfo SampleCommand {Additional details.}
1
% HelpxNS::MoreInfoFor SampleCommand
Additional details.
SEE ALSO
HelpxNS::EnterMoreInfo, HelpxNS::MoreInfoFor
